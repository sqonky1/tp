**CampusBridge is a desktop application designed for NUS students to organise and manage academic and campus-related contacts such as professors, teaching assistants, groupmates, and club members.** While it has a GUI, most of the user interactions happen using a CLI (Command Line Interface).

* If you are interested in using CampusBridge, head over to the [_Quick Start_ section of the **User Guide**](UserGuide.html#quick-start).
* If you are interested about developing CampusBridge, the [**Developer Guide**](DeveloperGuide.html) is a good place to start.
